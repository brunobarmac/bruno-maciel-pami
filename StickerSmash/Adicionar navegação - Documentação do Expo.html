Construa uma tela

Editar página

Neste tutorial, aprenda como usar componentes como o Pressable do React Native e o Image do Expo para construir uma tela.

Editar página

Neste capítulo, criaremos a primeira tela do aplicativo StickerSmash.

A tela acima exibe uma imagem e dois botões. O usuário do aplicativo pode selecionar uma imagem usando um dos dois botões. O primeiro botão permite que o usuário selecione uma imagem do seu dispositivo. O segundo botão permite que o usuário continue com uma imagem padrão fornecida pelo aplicativo.

Depois que o usuário selecionar uma imagem, poderá adicionar um adesivo a ela. Então, vamos começar a criar essa tela.

Assista: Criando uma tela no seu aplicativo universal Expo

1

Desmembre a tela

Antes de construirmos essa tela escrevendo código, vamos dividi-la em alguns elementos essenciais.

Existem dois elementos essenciais:

  • Uma grande imagem é exibida no centro da tela.
  • Existem dois botões na metade inferior da tela.

O primeiro botão contém vários componentes. O elemento pai fornece uma borda amarela e contém um ícone e componentes de texto dentro de uma linha.

Agora que dividimos a interface do usuário em partes menores, estamos prontos para começar a programar.

2

Exibir a imagem

Usaremos expo-imageuma biblioteca para exibir a imagem no aplicativo. Ela fornece um componente multiplataforma <Image>para carregar e renderizar uma imagem.

Pare o servidor de desenvolvimento pressionando Ctrl+ cno terminal. Em seguida, instale a expo-imagebiblioteca:

terminal
- npx expo install expo-image

O npx expo installcomando instalará a biblioteca e a adicionará às dependências do projeto no arquivo package.json .

Dica: Sempre que instalarmos uma nova biblioteca em nosso projeto, pare o servidor de desenvolvimento pressionando Ctrl+ cno terminal e, em seguida, execute o comando de instalação. Após a conclusão da instalação, podemos iniciar o servidor de desenvolvimento novamente executando o mesmo comando npx expo startna mesma janela do terminal.

O componente Imagem recebe a origem de uma imagem como seu valor. A origem pode ser um recurso estático ou uma URL. Por exemplo, a origem exigida do diretório assets/images é estática. Ela também pode vir da rede como uma uripropriedade.

Para usar o componente Image no arquivo app/(tabs)/index.tsx :

  1. Importar Imageda expo-imagebiblioteca.
  2. Crie uma PlaceholderImagevariável para usar o arquivo assets/images/background-image.png como sourcepropriedade no Imagecomponente.
app/(abas)/index.tsx
import { View, StyleSheet } from 'react-native';
import { Image } from 'expo-image';

const PlaceholderImage = require('@/assets/images/background-image.png');

export default function Index() {
  return (
    <View style={styles.container}>
      <View style={styles.imageContainer}>
        <Image source={PlaceholderImage} style={styles.image} />
      </View>
    </View>
  );
}

const styles = StyleSheet.create({
  container: {
    flex: 1,
    backgroundColor: '#25292e',
    alignItems: 'center',
  },
  imageContainer: {
    flex: 1,
  },
  image: {
    width: 320,
    height: 440,
    borderRadius: 18,
  },
});

3

Divida os componentes em arquivos.

Vamos dividir o código em vários arquivos à medida que adicionamos mais componentes a esta tela. Ao longo deste tutorial, usaremos o diretório `components` para criar componentes personalizados.

  1. Crie um diretório de componentes de nível superior e, dentro dele, crie o arquivo ImageViewer.tsx .
  2. Mova o código para exibir a imagem neste arquivo, juntamente com os imageestilos.
componentes/VisualizadorDeImagens.tsx
import { ImageSourcePropType, StyleSheet } from 'react-native';
import { Image } from 'expo-image';

type Props = {
  imgSource: ImageSourcePropType;
};

export default function ImageViewer({ imgSource }: Props) {
  return <Image source={imgSource} style={styles.image} />;
}

const styles = StyleSheet.create({
  image: {
    width: 320,
    height: 440,
    borderRadius: 18,
  },
});
Como o ImageViewer é um componente personalizado, estamos colocando-o em um diretório separado, em vez do diretório do aplicativo . Todos os arquivos dentro do diretório do aplicativo são arquivos de layout ou arquivos de rota. Para obter mais informações, consulte Componentes que não são de navegação ficam fora do diretório do aplicativo .

Importe ImageViewere utilize-o em app/(tabs)/index.tsx :

app/(abas)/index.tsx
import { StyleSheet, View } from 'react-native';

import ImageViewer from '@/components/ImageViewer';

const PlaceholderImage = require('@/assets/images/background-image.png');

export default function Index() {
  return (
    <View style={styles.container}>
      <View style={styles.imageContainer}>
        <ImageViewer imgSource={PlaceholderImage} />
      </View>
    </View>
  );
}

const styles = StyleSheet.create({
  container: {
    flex: 1,
    backgroundColor: '#25292e',
    alignItems: 'center',
  },
  imageContainer: {
    flex: 1,
  },
});
O que é a @declaração de importação?

O @símbolo é um alias de caminho personalizado para importar componentes personalizados e outros módulos em vez de caminhos relativos. O Expo CLI o configura automaticamente no arquivo tsconfig.json .

4

Crie botões usando o Pressable.

O React Native inclui alguns componentes diferentes para lidar com eventos de toque, mas <Pressable>é recomendado por sua flexibilidade. Ele pode detectar toques únicos, pressionamentos longos, disparar eventos separados quando o botão é pressionado e liberado, e muito mais.

No projeto, precisamos criar dois botões. Cada um tem um estilo e um rótulo diferentes. Vamos começar criando um componente reutilizável para esses botões. Crie um arquivo Button.tsx dentro do diretório components com o seguinte código:

componentes/Botão.tsx
import { StyleSheet, View, Pressable, Text } from 'react-native';

type Props = {
  label: string;
};

export default function Button({ label }: Props) {
  return (
    <View style={styles.buttonContainer}>
      <Pressable style={styles.button} onPress={() => alert('You pressed a button.')}>
        <Text style={styles.buttonLabel}>{label}</Text>
      </Pressable>
    </View>
  );
}

const styles = StyleSheet.create({
  buttonContainer: {
    width: 320,
    height: 68,
    marginHorizontal: 20,
    alignItems: 'center',
    justifyContent: 'center',
    padding: 3,
  },
  button: {
    borderRadius: 10,
    width: '100%',
    height: '100%',
    alignItems: 'center',
    justifyContent: 'center',
    flexDirection: 'row',
  },
  buttonLabel: {
    color: '#fff',
    fontSize: 16,
  },
});

O aplicativo exibe um alerta quando o usuário toca em qualquer um dos botões na tela. Isso acontece porque o componente <Pressable>chama alert()sua onPresspropriedade. Vamos importar este componente para o arquivo app/(tabs)/index.tsx e adicionar estilos para o <View>elemento que encapsula esses botões:

app/(abas)/index.tsx
import { View, StyleSheet } from 'react-native';

import Button from '@/components/Button';
import ImageViewer from '@/components/ImageViewer';

const PlaceholderImage = require("@/assets/images/background-image.png");

export default function Index() {
  return (
    <View style={styles.container}>
      <View style={styles.imageContainer}>
        <ImageViewer imgSource={PlaceholderImage} />
      </View>
      <View style={styles.footerContainer}>
        <Button label="Choose a photo" />
        <Button label="Use this photo" />
      </View>
    </View>
  );
}

const styles = StyleSheet.create({
  container: {
    flex: 1,
    backgroundColor: '#25292e',
    alignItems: 'center',
  },
  imageContainer: {
    flex: 1,
    paddingTop: 28,
  },
  footerContainer: {
    flex: 1 / 3,
    alignItems: 'center',
  },
});

5

Aprimore o componente de botão reutilizável.

O botão "Escolher uma foto" requer um estilo diferente do botão "Usar esta foto", então adicionaremos uma nova propriedade de tema ao botão que nos permitirá aplicar um primarytema. Este botão também possui um ícone antes do rótulo. Usaremos um ícone da @expo/vector-iconsbiblioteca.

Para carregar e exibir o ícone no botão, vamos usar FontAwesomea biblioteca. Modifique o arquivo components/Button.tsx para adicionar o seguinte trecho de código:

componentes/Botão.tsx
import { StyleSheet, View, Pressable, Text } from 'react-native';
import FontAwesome from '@expo/vector-icons/FontAwesome';

type Props = {
  label: string;
  theme?: 'primary';
};

export default function Button({ label, theme }: Props) {
  if (theme === 'primary') {
  return (
      <View
        style={[
          styles.buttonContainer,
          { borderWidth: 4, borderColor: '#ffd33d', borderRadius: 18 },
        ]}>
        <Pressable
          style={[styles.button, { backgroundColor: '#fff' }]}
          onPress={() => alert('You pressed a button.')}>
          <FontAwesome name="picture-o" size={18} color="#25292e" style={styles.buttonIcon} />
          <Text style={[styles.buttonLabel, { color: '#25292e' }]}>{label}</Text>
        </Pressable>
      </View>
    );
  }

  return (
    <View style={styles.buttonContainer}>
      <Pressable style={styles.button} onPress={() => alert('You pressed a button.')}>
        <Text style={styles.buttonLabel}>{label}</Text>
      </Pressable>
    </View>
  );
}

const styles = StyleSheet.create({
  buttonContainer: {
    width: 320,
    height: 68,
    marginHorizontal: 20,
    alignItems: 'center',
    justifyContent: 'center',
    padding: 3,
  },
  button: {
    borderRadius: 10,
    width: '100%',
    height: '100%',
    alignItems: 'center',
    justifyContent: 'center',
    flexDirection: 'row',
  },
  buttonIcon: {
    paddingRight: 8,
  },
  buttonLabel: {
    color: '#fff',
    fontSize: 16,
  },
});

Vamos aprender o que o código acima faz:

  • O botão do tema principal usa estilos embutidos , que substituem os estilos definidos StyleSheet.create()com um objeto passado diretamente na stylepropriedade.
  • O <Pressable>componente no tema principal usa uma backgroundColorpropriedade com um valor #fffpara definir o fundo do botão como branco. Se adicionarmos essa propriedade ao styles.button, o valor da cor de fundo será definido tanto para o tema principal quanto para o tema sem estilo.
  • Os estilos embutidos usam JavaScript e substituem os estilos padrão para um valor específico.

Agora, modifique o arquivo app/(tabs)/index.tsx para usar a theme="primary"propriedade no primeiro botão.

app/(abas)/index.tsx
import { View, StyleSheet } from 'react-native';

import Button from '@/components/Button';
import ImageViewer from '@/components/ImageViewer';

const PlaceholderImage = require('@/assets/images/background-image.png');

export default function Index() {
  return (
    <View style={styles.container}>
      <View style={styles.imageContainer}>
        <ImageViewer imgSource={PlaceholderImage} />
      </View>
      <View style={styles.footerContainer}>
        <Button theme="primary" label="Choose a photo" />
        <Button label="Use this photo" />
      </View>
    </View>
  );
}

const styles = StyleSheet.create({
  container: {
    flex: 1,
    backgroundColor: '#25292e',
    alignItems: 'center',
  },
  imageContainer: {
    flex: 1,
  },
  footerContainer: {
    flex: 1 / 3,
    alignItems: 'center',
  },
});
